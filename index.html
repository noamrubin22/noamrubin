Gerrit

chat with random people, near to you!
go to web application see code

A full-stack MERN web application that allows users to chat, only while being in the same location. An environment where neighbors could get in touch, tourists could ask for area specific restaurant tips and people that feel alone could reach out to others near to them. No one can enter the chat box while not being in the specific area it concerns.

Technologies

React - MongoDB - Mongoose - Express - socket.io - Node.js - CSS - HTML
