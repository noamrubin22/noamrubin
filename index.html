Gerrit

chat with random people, near to you!

React * MongoDB * Mongoose * Express * socket.io * Node.js * CSS * HTML

go to web application
see code

Context

For the last project of the Ironhack bootcamp we had to create a full-stack MERN (MongoDB, Express, React, Node.js) web application. In teams of 3, we had one week to make it happen. We decided to create a mobile web application that allows users to chat with others only while being in the same location. The idea behind it was to create an environment where neighbors could talk to each other (for example if someone lost their cat/keys), tourists could ask for restaurant tips while being in a specific area and people in- or outside a club can communicate with each other concerning the queue, djs and vibes. Users create a profile but can still stay anonymous, giving them a bit more freedom to express themselves. No one can enter the chat box while not being in the specific location it concerns.

Development

The chat was implemented with socket.io.
